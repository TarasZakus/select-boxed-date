# select-boxed-date
Provides convenient way to select a date using 3 select-boxes.

Defaults:
		rangeFrom: "-200y",
		rangeTo: "-12y",
		rangeRound: true,
		labelDay: "Day",
		labelMonth: "Month",
		labelYear: "Year",
		format: "m-d-Y",
		mapping: "day|month|year",
		onSelectBoxesCreated: function() {/* Apply some styling or anything else. */},
		onSelectBoxOptionsChanged: function(selectBox) {}
